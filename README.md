## Overview
Ebola Info Center is a responsive informational website designed to provide clear, accessible, and trustworthy information about Ebola outbreaks. The experience focuses on helping users quickly understand symptoms, prevention methods, and transmission risks without overwhelming them with complex medical language.

This project was created as part of my UX/UI design and web design portfolio, emphasizing user-centered design and clarity in high-stakes informational experiences.
